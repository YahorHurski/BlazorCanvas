| browser -> POST /logout | Untrusted, potentially cross-site form POST reaches the sign-out endpoint |
| every request -> cookie middleware | Session cookie validated on every request; `[Authorize]` + `LoginPath` gate protected pages |
| app -> PostgreSQL | Username lookup/compare crosses into the `users` table |
