<objective>
Switch the live circuit to v1.11 rows while the additive schema remains available: render local
shapes inside translated/rotated groups, draw through the gateway, drag by position only, and
preserve all v1.1 cross-tab and failure outcomes with UUID identity.
</objective>

<threat_model>
| STRIDE | Threat | Disposition |
|---|---|---|
| Spoofing | A sync message names another user's UUID. | Mitigated: notifier user buckets plus receiver mutations limited to rows already loaded for its resolved canvas. |
| Tampering | Move carries shape data or resurrects a deleted row. | Mitigated: move/rollback carry only UUID and x/y; unknown UUID is ignored (D-40). |
| Repudiation | Optimistic glides diverge after failed save. | Mitigated: retained original x/y broadcasts rollback; acknowledged modal reload publishes canonical snapshot. |
| Information disclosure | Draw relays untrusted JSON. | Mitigated: draw relays repository-returned canonical row after gateway validation. |
| Denial of service | Pointer storms flood tabs. | Mitigated: 50ms throttle and one final trailing broadcast; no pointer-move database write. |
| Elevation of privilege | Remote delivery mutates a circuit during drag/disposal. | Mitigated: D-54 discard occurs first; sender filter and existing InvokeAsync disposal handling remain. |
</threat_model>
